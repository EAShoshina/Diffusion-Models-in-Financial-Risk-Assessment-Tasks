## Аннотация

Программа реализует метод оценки рыночных рисков портфеля ликвидных ценных бумаг на основе диффузионной вероятностной модели (Denoising Diffusion Probabilistic Model, DDPM) с нейросетевым денойзером на архитектуре трансформера. Программа осуществляет автоматизированную загрузку и предобработку биржевых данных, построение рядов доходностей, обучение нейросетевой модели, генерацию сценариев доходностей и расчёт показателей рыночного риска Value‑at‑Risk (VaR) и Expected Shortfall (ES) на уровне доверия 99%. Верификация результатов осуществляется посредством статистических процедур бэктестинга, включая критерии Купика и Кристофферсена для оценки частоты нарушений VaR, а также тест Дибольда–Мариано для сравнения прогностической точности с альтернативными моделями. Программа апробирована на данных Московской биржи по акциям ПАО «Сбербанк России», ПАО «ЛУКОЙЛ» и ПАО «Газпром» за период 2015–2024 годов. По результатам апробации модель обеспечивает долю нарушений VaR порядка 0,67% при целевом уровне 1,0%, что соответствует «зелёной» зоне регуляторного стандарта Basel III.

## Методология

### 1. Диффузионная вероятностная модель (DDPM)

Реализована схема Denoising Diffusion Probabilistic Model с **косинусным расписанием шума** (Nichol & Dhariwal, 2021).

**Прямой процесс:**

$$x_t = \sqrt{\bar{\alpha}_t} \cdot x_0 + \sqrt{1 - \bar{\alpha}_t} \cdot \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, I)$$

**Косинусное расписание:**

$$f(t) = \cos^2\!\left(\frac{t/T + s}{1 + s} \cdot \frac{\pi}{2}\right), \quad s = 0.008$$

$$\bar{\alpha}_t = \frac{f(t)}{f(0)}$$

Ключевые значения при \(T = 50\): \(\bar{\alpha}_{T/4} = 0{,}8356\), \(\bar{\alpha}_{T/2} = 0{,}4627\), \(\bar{\alpha}_T \approx 0\).

**Параметризация:** \(\varepsilon\)-prediction. Выбор обусловлен равномерным взвешиванием всех шагов диффузии, что обеспечивает качественное воспроизведение хвостовых событий.

**Обратный процесс:** итеративная денойзинговая выборка \(T \to 0\) с клиппингом \(\hat{x}_0 \in [-10\sigma, +10\sigma]\).

---

### 2. Архитектуры денойзеров

#### TransformerDenoiser (основная модель)

Pre-LN Transformer (`norm_first=True`) с контекстным окном W=30.

| Компонент              | Параметр                                |
|------------------------|-----------------------------------------|
| Входная последовательность | [c_{t−W}, ..., c_{t−1}, x_t] — W+1 токенов |
| Проекция токенов       | Linear(1 → d_model)                     |
| Позиционное кодирование | Синусоидальное (Vaswani et al., 2017)   |
| Эмбеддинг шага t       | nn.Embedding(T, d_model), добавляется к x_t |
| Encoder layers         | Pre-LN TransformerEncoderLayer × L      |
| Голова                 | LayerNorm → Linear(d_model → 1)         |
| Выход                  | Скалярная оценка шума ε̂                |
| **Параметров**         | **153 473**                             |

> Механизм self-attention напрямую моделирует нелокальные зависимости в окне W, соответствующие ARCH-эффекту в финансовых временных рядах.

#### MLPDenoiser (baseline Ablation A)

Трёхслойный MLP без контекстного окна. Намеренно игнорирует c_t — используется для количественной оценки вклада контекста.

| Компонент       | Параметр                         |
|-----------------|----------------------------------|
| Вход            | [x_t, step_emb(t)]               |
| Скрытые слои    | 3 × [Linear(256) → SiLU → LN]   |
| **Параметров**  | **212 225**                      |

## Статистические тесты и метрики

В коде реализованы следующие процедуры верификации:

| Тест / Метрика                  | Функция                | Гипотеза / Назначение                                   |
|---------------------------------|------------------------|---------------------------------------------------------|
| Тест Купика (POF)               | `kupiec_test()`        | H₀: доля нарушений = 1 − α                             |
| Тест Кристофферсена (CC)        | `christoffersen_test()`| H₀: нарушения независимы и доля = 1 − α                |
| Тест McNeil–Frey                | `mcneil_frey_test()`   | H₀: ES откалиброван                                    |
| Тест Дибольда–Мариано           | `diebold_mariano_test()`| H₀: равная точность прогнозов по tick loss            |
| KS PIT-тест                     | `stats.kstest(..., 'uniform')` | H₀: u_t ~ Uniform(0,1) — корректная калибровка |
| Bootstrap CI (1000 итераций)    | `bootstrap_ci()`       | 95% CI для доли нарушений и tick loss                  |
| KL-дивергенция                  | `kl_sym()`, `kl_tail()`| Расстояние между модельным и эмпирическим распределениями |
| Quantile loss (Pinball)         | `tick_loss()`          | ρ_α(r, q) = (α − 1{r<q}) · (r − q)                   |
| CRPS                            | `crps_empirical()`, `crps_normal()` | Непрерывный рейтинговый показатель     |
| Basel III Traffic Light         | `basel_traffic_light()`| Классификация: зелёная / жёлтая / красная зона         |
